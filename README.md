# cppgraphics
compile with
```g++ main.cpp -lgdi32```

lightweight, mostly dependency free (uses stb_image.h)  
software rendered framebuffer  
32bit pixel buffer  
png sprite loading  
alpha blended sprite drawing  
scaled sprite drawing  
delta-time based update loop for framerate independent movement and animations using high resolution timers  
full per pixel control  
text drawing with my 8x8 font  
minimal win32 backend  
  
use freely for anything youd like only credit me if you use my font
